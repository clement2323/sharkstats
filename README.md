## Shark incidents by State Dashboard

A Streamlit dashboard to visualize Shark-incident data across Australian states. The dashboard allows users to:

- Explore incidents by state, month, hour, and year.
- Compare a selected state against the peer average of other states.
- View incidents interactively on a Folium map.
- Customize selection of states and year range for dynamic analysis.

### Features:

- Clean, responsive interface using Streamlit.
- Interactive line charts powered by Altair.
- Consistent color scheme for states and peer averages.
- Designed for easy comparison of temporal trends in incidents.

### Technologies used:
Python, Streamlit, Pandas, Altair, Folium
